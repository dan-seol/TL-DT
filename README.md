# TL;DT (Too Long; Didn't Type)

**Description:**
</br> Converts speech to code

## Set Up

**Set up Python 3.6 environment in Anaconda:**
</br> *Open Anaconda Prompt and enter the following commands:*

```
conda create --name python36 python=3.6
activate python36
conda install 
```
